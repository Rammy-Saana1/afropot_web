Overview

AFROPOT is a web-based Nigerian food ordering platform.
Customers can browse meals, place orders, and make online payments through a simple web interface.

Features

User registration and login

Browse food menus

Add items to cart

Place food orders

Online payment (Stripe)

View past orders

Project Scope

Included: Customer registration, food browsing, ordering, payment, and order tracking.

Technologies

Frontend: HTML, CSS, JavaScript

Backend: Node.js

Database: NoSQL

Tools: Git, npm, Stripe
